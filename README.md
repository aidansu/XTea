# XTea

TEA加密算法改进版

> 每次操作可以处理8个字节数据
KEY为16字节,应为包含4个int型数的int[]，一个int为4个字节
加密解密轮数应为8的倍数，推荐加密轮数为64轮，本程序使用32轮

#### 不同开发语言版本
* XTea-Go     : go版本
* XTea-Java   : Java版本  (Java 8)
* XTea-Python : Python版本
